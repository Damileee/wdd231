SwiftFit – Project Site Plan

1. Site Name

SwiftFit — This name was chosen to reflect a fast and effective approach to fitness for beginners. It communicates a sense of energy, movement, and simplicity.

Example domain: swiftfitguide.com

2. Site Purpose

The purpose of SwiftFit is to provide a clear and simple starting point for people new to fitness. The website offers:

  • Beginner-friendly workout routines (from a local JSON file)
  • Healthy recipe suggestions (from a remote API)
  • Encouraging tips and motivation for new fitness journeys
  • Dynamic features using JavaScript and browser APIs

3. Scenarios

  • “What are some workouts I can do at home with no equipment?”
  • “Can I find healthy meal ideas to support my beginner fitness goals?”

4. Color Schema

  • Primary Color: #007acc — used for headers, navigation, and brand elements
  • Accent Color: #28a745 — used for call-to-actions, highlights, and buttons

5. Typography

  • Headings: Roboto Bold — clean and modern, gives structure to the page
  • Body Text: Open Sans Regular — easy to read and neutral for long content

This site plan document already applies these fonts live via Google Fonts.
